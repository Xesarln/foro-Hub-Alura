## FUNCIONALIDADES

## POST /topicos/login:

## POST /topicos: 
Permite a los usuarios crear nuevos topicos en el foro con su token generado.
<img src="img\Registro Topico.png"/>
## GET /topicos/listado: 
Muestra una lista de todos los topicos disponibles.
<img src="img\LISTADO.png"/>
## PUT /topicos
Permite la actualizacion de la informacion de un topico existente.
<img src="img\ACTUALIZAR TOPICO.png"/>
## DELETE /topicos/{id}: 
Elimina topicos no deseados del foro.
<img src="img\DELETE.png"/>
